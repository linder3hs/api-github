># Api - github

## Steps for deploy and test 
  
1) Download project
 `git clone https://github.com/linder3hs/api-github`

2) Move to directory
 `cd api-github`

3) Install dependencies
`npm i`

4) Run local server
  `npm start`

5) Run unit testing 
  `npm test`
